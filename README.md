# Salary Estimation using K-Nearest Neighbour (KNN)

This project predicts whether an employee earns more than 50K per year using the K-Nearest Neighbour (KNN) algorithm.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Dataset
- salary.csv
- Features: Age, Education Level, Capital Gain, Working Hours
- Target: Income (<=50K or >50K)

## How to Run
1. Install dependencies:
   pip install pandas numpy scikit-learn matplotlib

2. Run the program:
   python KNN.py

3. Enter employee details when prompted.

## Output
- Salary prediction
- Accuracy score
- Confusion matrix
